# 🎮 Hangman Game

An interactive Hangman web application developed using Python and Flask.

## Features

* 🎯 Guess letters
* ❤️ Lives system
* 🏆 Score system
* 🎨 Difficulty levels
* 📊 Progress bar
* 💡 Hints
* 🔄 Play Again button
* 🎮 Modern UI

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript

## How to Run

Install Flask:

```bash
pip install flask
```

Run:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
